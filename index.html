Fred The Hitchhiker



Fred is a Discord bot that works like a hitchhiker: If it is invited to a Discord Server, it leaves the previous server it was in and joins the new one. Additionaly it changes it's profile picture everytime. The profile pictures are taken from https://arthurfindelair.com/thisnightskydoesnotexist/.
